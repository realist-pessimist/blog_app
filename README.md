# blog_app
Blog App

Flask TDD Aplication 

## Project setup
```
setup venv
pip install pytest
pip install flask
pip install pydantic
pip install email-validator
pip install jsonschema
```

### Run modules Tests
```
select Run/Debug configuration - pytests
```
### Run instumental Tests
terminal:
```
cd/blog_app
set FLASK_APP=blog/app.py
set FLASK_ENV=development
init_db.py
flask run
```
